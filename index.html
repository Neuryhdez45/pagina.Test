Que es Discord.js?

Discord.js es un modulo de Node.js Creado para interactuar con las API De discord mas facil.
Este modulo esta hecho para crear bots de discord mas facil.
Un poco mas de informacion sacada desde su pagina ↆ
Se necesita un enfoque mucho más orientado a objetos que la mayoría de las otras bibliotecas JS Discord, lo que hace que el código de su bot sea mucho más ordenado y fácil de comprender.

Discord.js usa orientacion de objetos, y modulo de funcionamientos faciles.
